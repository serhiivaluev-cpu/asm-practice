int 0x80 — це інструкція програмного переривання в операційній системі Linux для архітектури x86 (32-bit). Вона використовується для виконання системних викликів, тобто звернення програми до ядра операційної системи.

Перед виконанням int 0x80 у регістри записуються необхідні значення:

EAX — номер системного виклику

EBX, ECX, EDX, ESI, EDI, EBP — параметри, які передаються цьому виклику

Найбільш поширені системні виклики

sys_exit — використовується для завершення роботи програми.

EBX — код завершення програми.

sys_write — записує дані у файл або виводить їх у консоль.

EBX — файловий дескриптор

ECX — адреса буфера з даними

EDX — довжина даних у байтах

sys_read — читає дані з файлу або з консолі.

EBX — файловий дескриптор (наприклад, 0 — стандартний ввід stdin)

ECX — адреса буфера для зчитування

EDX — кількість байтів, які потрібно прочитати

sys_open — відкриває файл.

EBX — ім’я файлу

ECX — режим відкриття файлу

EDX — права доступу

sys_close — закриває відкритий файл.

EBX — файловий дескриптор файлу
